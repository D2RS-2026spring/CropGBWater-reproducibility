# Processed data

This folder stores the processed data used to reproduce the main results of this course project.

The core processed table is:

four_crop_2020_water_consumption.csv

This table summarizes 2020 global water consumption for four major crops:

- Maize
- Soybean
- Rice
- Wheat

The water-consumption components include:

- Rainfed green water
- Irrigated green water
- Blue water
- Paddy flooding, only for Rice

The processed data are derived from the public CropGBWater output files. The purpose of providing this processed table is to make the course project lightweight and reproducible without requiring every user to download the full original Zenodo dataset.

The full original data source is documented in data/raw/README.md.
