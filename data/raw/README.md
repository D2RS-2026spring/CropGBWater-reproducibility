The original CropGBWater data and code are available from Zenodo:

https://doi.org/10.5281/zenodo.13901563

The reproduced paper is:

Global spatially explicit crop water consumption shows an overall increase of 9% for 46 agricultural crops from 2010 to 2020

Nature Food, 2025
DOI: https://doi.org/10.1038/s43016-025-01231-x

Because the original input and output files are large, this repository does not directly store the full raw dataset. Instead, the project provides processed four-crop 2020 summary data under data/processed/, which is sufficient to reproduce the main figures and tables in this course project.

Users who want to reproduce the full model workflow should download the original data from the Zenodo record above and follow the data organization described in the project README.
